# Dataset Options for Workshop Projects

Participants choose one dataset and build an app around it. Each dataset includes suggested project ideas.

## Provided Datasets (to be added)

### 1. Biodiversity / Species Observations
- **Source options**: GBIF, iNaturalist, local survey data
- **Columns**: species, lat, lon, date, observer, habitat type
- **Project ideas**:
  - Interactive species distribution map
  - Biodiversity hotspot explorer
  - Seasonal observation patterns dashboard
  - Species richness calculator by area

### 2. Water Quality / Limnology
- **Source options**: Lake Winnipeg monitoring data, provincial databases
- **Columns**: station, date, temp, pH, dissolved_oxygen, nutrients, chlorophyll
- **Project ideas**:
  - Lake health dashboard with time series
  - Water quality comparison tool across stations
  - Nutrient loading visualizer
  - Algal bloom risk indicator

### 3. Conservation / Protected Areas
- **Source options**: IUCN, Canadian Protected Areas Database
- **Columns**: name, designation, area_km2, year_established, biome, species_count
- **Project ideas**:
  - Protected area explorer with maps
  - Conservation gap analysis tool
  - Biodiversity representation dashboard

### 4. Climate / Environmental Data
- **Source options**: Environment Canada, ERA5, local weather stations
- **Columns**: station, date, temp, precip, wind, humidity
- **Project ideas**:
  - Climate trends explorer
  - Growing season calculator
  - Extreme weather event tracker

### 5. Genomics / Bioinformatics (Advanced)
- **Source options**: eDNA survey results, amplicon data summaries
- **Columns**: sample_id, site, date, taxon, read_count, relative_abundance
- **Project ideas**:
  - Community composition explorer
  - Diversity index calculator
  - Sample comparison tool
  - Taxonomic abundance visualizer

### 6. Ecosystem Services
- **Source options**: InVEST model outputs, land use data
- **Columns**: region, land_use, carbon_storage, water_filtration, pollination_value
- **Project ideas**:
  - Ecosystem service valuation dashboard
  - Land use change impact simulator
  - Nature's contribution to people explorer

### 7. GIS / Land Use
- **Source options**: Land cover databases, remote sensing classifications
- **Columns**: region, lat, lon, land_class, area, change_year
- **Project ideas**:
  - Land use change viewer
  - Habitat fragmentation analyzer
  - Deforestation/restoration tracker

## Bring Your Own Data
Participants are encouraged to bring their own research datasets! The AI assistant can help build an app around any tabular data.

## Dataset Requirements
For the workshop to work smoothly, datasets should:
- Be in CSV format (or easily convertible)
- Have at least 100 rows
- Include at least one categorical and one numerical column
- Ideally include geographic coordinates for mapping
- Be shareable (no confidential or restricted data)
