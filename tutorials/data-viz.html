Data Visualization with Plotly

üìñ 20 min read ¬∑ üü° Intermediate

Good visualizations make your data tell a story. Plotly creates interactive charts that users can zoom, pan, hover over, and explore ‚Äî perfect for ecological data.

Why Plotly?

  • Interactive ‚Äî zoom, pan, hover for details, click to select
  • Beautiful defaults ‚Äî looks professional out of the box
  • Streamlit integration ‚Äî works seamlessly with st.plotly_chart()
  • Maps included ‚Äî built-in geographic visualization

Chart Types for Ecology

Scatter Plots ‚Äî Relationships Between Variables

Perfect for: morphometric comparisons, environmental gradients, correlation exploration.

Prompt

"Create a scatter plot of body mass vs flipper length, colored by species. Add trendlines for each species. Include proper axis labels with units."

Python 
fig = px.scatter(
    df, x="flipper_length_mm", y="body_mass_g",
    color="species",
    trendline="ols",
    labels={
        "flipper_length_mm": "Flipper Length (mm)",
        "body_mass_g": "Body Mass (g)"
    },
    title="Body Mass vs Flipper Length by Species"
)
st.plotly_chart(fig, use_container_width=True)

Bar Charts ‚Äî Comparisons Across Categories

Perfect for: species counts, habitat comparisons, treatment effects.

Prompt

"Create a grouped bar chart comparing mean body mass across species, grouped by island. Add error bars showing standard deviation."

Python 
# Compute summary stats
summary = df.groupby(["species", "island"])["body_mass_g"].agg(["mean", "std"]).reset_index()

fig = px.bar(
    summary, x="species", y="mean", color="island",
    error_y="std", barmode="group",
    labels={"mean": "Mean Body Mass (g)", "species": "Species"},
    title="Body Mass by Species and Island"
)
st.plotly_chart(fig, use_container_width=True)

Box Plots ‚Äî Distribution Comparison

Perfect for: comparing measurements across groups, identifying outliers.

Python 
fig = px.box(
    df, x="species", y="bill_length_mm", color="species",
    points="all",  # Show individual data points
    labels={"bill_length_mm": "Bill Length (mm)"},
    title="Bill Length Distribution by Species"
)
st.plotly_chart(fig, use_container_width=True)

Histograms ‚Äî Distribution of a Single Variable

Perfect for: understanding data spread, identifying patterns.

Python 
fig = px.histogram(
    df, x="body_mass_g", color="species",
    nbins=30, opacity=0.7,
    marginal="box",  # Adds box plot on top
    labels={"body_mass_g": "Body Mass (g)"},
    title="Distribution of Body Mass"
)
st.plotly_chart(fig, use_container_width=True)

Line Charts ‚Äî Time Series

Perfect for: monitoring data, population trends, seasonal patterns.

Python 
fig = px.line(
    df, x="date", y="temperature",
    color="station",
    labels={"temperature": "Temperature (¬∞C)", "date": "Date"},
    title="Water Temperature Over Time"
)
fig.update_layout(hovermode="x unified")
st.plotly_chart(fig, use_container_width=True)

Heatmaps ‚Äî Correlation Matrices

Perfect for: identifying relationships between many variables at once.

Python 
import plotly.figure_factory as ff

# Select numeric columns
numeric_cols = df.select_dtypes(include="number").columns
corr = df[numeric_cols].corr()

fig = px.imshow(
    corr,
    labels=dict(color="Correlation"),
    color_continuous_scale="RdBu_r",
    zmin=-1, zmax=1,
    title="Variable Correlation Matrix"
)
st.plotly_chart(fig, use_container_width=True)
