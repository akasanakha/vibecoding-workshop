Streamlit Basics

üìñ 15 min read ¬∑ üü¢ Beginner

Streamlit turns Python scripts into interactive web apps. It's the fastest way to go from data to deployed dashboard, and it's our recommended tool for this workshop.

Why Streamlit?

  • Minimal learning curve ‚Äî if you've seen a Python script, you can read Streamlit code
  • No web development knowledge needed ‚Äî no HTML, CSS, or JavaScript required
  • Built for data ‚Äî native support for dataframes, charts, and maps
  • Free deployment ‚Äî Streamlit Cloud hosts your app for free
  • AI understands it ‚Äî every AI coding assistant knows Streamlit well

The Anatomy of a Streamlit App

Here's the simplest possible Streamlit app:

app.py 
import streamlit as st

st.title("Hello, Ecology! üåø")
st.write("This is my first Streamlit app.")

That's it. Two lines of meaningful code, and you have a web app.

A Real Ecology Example

Here's a more realistic app that loads data and creates an interactive visualization:

app.py 
import streamlit as st
import pandas as pd
import plotly.express as px

# Page config
st.set_page_config(
    page_title="Penguin Explorer",
    page_icon="üêß",
    layout="wide"
)

# Title
st.title("üêß Palmer Penguins Explorer")
st.write("Explore morphological measurements of penguins in Antarctica.")

# Load data
@st.cache_data
def load_data():
    url = "https://raw.githubusercontent.com/allisonhorst/palmerpenguins/main/inst/extdata/penguins.csv"
    return pd.read_csv(url)

df = load_data()

# Sidebar filters
st.sidebar.header("Filters")
species = st.sidebar.multiselect(
    "Species",
    options=df["species"].unique(),
    default=df["species"].unique()
)
island = st.sidebar.multiselect(
    "Island",
    options=df["island"].unique(),
    default=df["island"].unique()
)

# Filter data
filtered = df[
    (df["species"].isin(species)) &
    (df["island"].isin(island))
]

# Show metrics
col1, col2, col3 = st.columns(3)
col1.metric("Total Penguins", len(filtered))
col2.metric("Species", filtered["species"].nunique())
col3.metric("Islands", filtered["island"].nunique())

# Scatter plot
fig = px.scatter(
    filtered,
    x="bill_length_mm",
    y="bill_depth_mm",
    color="species",
    title="Bill Length vs Bill Depth",
    labels={
        "bill_length_mm": "Bill Length (mm)",
        "bill_depth_mm": "Bill Depth (mm)"
    }
)
st.plotly_chart(fig, use_container_width=True)

# Data table
st.subheader("Raw Data")
st.dataframe(filtered)

Key Streamlit Components

Text & Layout

Python 
st.title("Main Title")           # Big heading
st.header("Section Header")      # Medium heading
st.subheader("Subsection")       # Smaller heading
st.write("Regular text")         # Paragraph text
st.markdown("**Bold** and *italic*")  # Markdown support

# Columns for side-by-side layout
col1, col2 = st.columns(2)
with col1:
    st.write("Left side")
with col2:
    st.write("Right side")

# Tabs for organized content
tab1, tab2 = st.tabs(["Charts", "Data"])
with tab1:
    st.write("Charts go here")
with tab2:
    st.write("Data table goes here")

Data Display

Python 
st.dataframe(df)              # Interactive table
st.table(df.head())           # Static table
st.metric("Label", 42, "+5")  # Metric with delta
st.json({"key": "value"})    # JSON display

Charts (Plotly)

Python 
import plotly.express as px

# Scatter plot
fig = px.scatter(df, x="col_x", y="col_y", color="category")
st.plotly_chart(fig, use_container_width=True)

# Bar chart
fig = px.bar(df, x="category", y="count", color="group")
st.plotly_chart(fig)

# Line chart (time series)
fig = px.line(df, x="date", y="value", color="station")
st.plotly_chart(fig)

# Histogram
fig = px.histogram(df, x="measurement", nbins=30, color="species")
st.plotly_chart(fig)

# Box plot
fig = px.box(df, x="species", y="body_mass_g", color="species")
st.plotly_chart(fig)

Maps

Python 
# Simple map (requires 'latitude' and 'longitude' columns)
st.map(df)

# Advanced map with Plotly
fig = px.scatter_mapbox(
    df,
    lat="latitude",
    lon="longitude",
    color="species",
    size="abundance",
    hover_name="site_name",
    mapbox_style="open-street-map",
    zoom=5,
    title="Species Observations"
)
st.plotly_chart(fig)

User Input (Sidebar Filters)

Python 
# Dropdown (single select)
choice = st.sidebar.selectbox("Select species", df["species"].unique())

    "Body mass range (g)",

    value=(2000, 6500)

# Date input
date = st.sidebar.date_input("Select date")

# Text input

# Checkbox
show_raw = st.sidebar.checkbox("Show raw data", value=False)
if show_raw:
    st.dataframe(df)
